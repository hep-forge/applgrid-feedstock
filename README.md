Feedstock for [applgrid](https://applgrid.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

The APPLgrid project provides a fast and flexible way to reproduce the results of full NLO calculations

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `1.6.35` |
| linux-arm64 (`linux-aarch64`) | ✅ `1.6.35` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge applgrid
```
