[![Build AMD64](https://github.com/hep-forge/applgrid-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/applgrid-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/applgrid-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/applgrid-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [applgrid](https://applgrid.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).

The APPLgrid project provides a fast and flexible way to reproduce the results of full NLO calculations

## Install

```bash
conda install -c hep-forge -c conda-forge applgrid
```
